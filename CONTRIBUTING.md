# Contributing

Thanks for your interest in improving this project.

## Development Setup

1. Install Flutter stable (`3.38.x` recommended).
2. Clone this repository.
3. Run:
   - `flutter pub get`
   - `flutter analyze`
   - `flutter test`

## Branching

- Create a feature branch from `main`.
- Keep Pull Requests focused and small.
- Use clear commit messages.

## Pull Request Checklist

- [ ] Code builds and runs locally.
- [ ] `flutter analyze` passes.
- [ ] `flutter test` passes.
- [ ] Documentation is updated if behavior changed.

## Code Style

- Follow `flutter_lints`.
- Keep code readable and avoid unrelated refactors in the same PR.

## Reporting Bugs

Please use the Bug Report issue template and include reproduction steps and logs.
